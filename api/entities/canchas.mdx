## Overview

The `CE_Canchas` entity represents a sports court in the system. It contains information about the court's name, type, pricing, and availability status.

## Properties

<ParamField name="IdCancha" type="int" required>
  Unique identifier for the sports court
</ParamField>

<ParamField name="Nombre" type="string">
  Name of the sports court
</ParamField>

<ParamField name="Tipo" type="string">
  Type of sports court (e.g., football, basketball, tennis)
</ParamField>

<ParamField name="PrecioPorHora" type="decimal" required>
  Price per hour for renting the court
</ParamField>

<ParamField name="Estado" type="string">
  Current status of the court (e.g., "Disponible", "En mantenimiento")
</ParamField>

## Entity Definition

## Usage Example

```csharp
// Creating a new court
var cancha = new CE_Canchas
{
    IdCancha = 1,
    Nombre = "Cancha Fútbol 1",
    Tipo = "Fútbol",
    PrecioPorHora = 50.00m,
    Estado = "Disponible"
};

// Using the ViewModel

    ListaCanchas = new List<CE_Canchas>()
```
